setMetaTag(), removeMetaTag()


setMetaTag() adds or updates a <meta> tag in <head>. Supports both name= and property= attributes (e.g. Open Graph).
removeMetaTag() removes a <meta> tag by its name or property value.
Open DevTools → Elements → <head> to observe the changes.


Tags set on page load:

  • setMetaTag('robots', 'index, follow')
  • setMetaTag('og:title', 'DoDo Framework Examples') — uses property= attribute
  • setMetaTag('og:image', '/public/img/logo.png')
