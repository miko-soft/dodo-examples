loadJS(), unloadJS, unloadAllJS

The example loads "jQuery", "Notify" and "Sweetalert" libraries.
NOTICE: The unloadJS() will remove SCRIPT tag but it will not unload JS file from the browser memory. So it will stay active.

unloadJS() in __destroy() hook

We can put "this.unloadAllJS();" in the destroy hook if we want to unload JS files which are not needed on other pages.
It will remove JS files from DOM but it will be still active in browser's memory.
If you want to remove it from the memory too, use corelib.navig.reload() to reload the page.

To test this, click on /controller/hooks and watch Elements in Chrome DevTools. Return to this page and click POPUP button. The popup will work because JS libs are not removed from browser memory.
