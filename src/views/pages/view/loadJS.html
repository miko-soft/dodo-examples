loadJS(), unloadJS, unloadAllJS

This example loads jQuery, Notify, and SweetAlert libraries.
Note: unloadJS() removes the SCRIPT tag but does not unload the file from browser memory — it remains active.

unloadJS() in __destroy() hook

You can call this.unloadAllJS() in the __destroy() hook to remove JS files that are not needed on other pages.
This removes them from the DOM but not from browser memory.
To fully remove them, call corelib.navig.reload() to reload the page.

To test this: navigate to /controller/hooks and inspect Elements in Chrome DevTools. Return to this page and click POPUP — it will still work because the JS libraries were not removed from browser memory.
