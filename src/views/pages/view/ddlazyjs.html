dd-lazyjs

The attribute dd-lazyjs will load JS lazily. To make this work simply put dd-lazyjs attribute in the SCRIPT tag.
There are three options: dd-lazyjs="--after__loader" or dd-lazyjs="--after__rend" or dd-lazyjs="--after__postrend"

To test when the lazy loading occurs, enable debugging by setting: this.$debugOpts.ddLazyjs = true;. This will provide detailed logs about the lazy loading process.


NOTICE:
will work → <script src="/assets/js/jq-test.js" dd-lazyjs="--after__rend"></script>
will also work → <script src="/assets/js/jq-test-module.js" type="module" dd-lazyjs="--after__rend"></script>
