dd-lazyjs

The dd-lazyjs attribute defers JavaScript loading. Add dd-lazyjs to a SCRIPT tag to enable it.
Three options control when the script is loaded: dd-lazyjs="--after__loader", dd-lazyjs="--after__rend", or dd-lazyjs="--after__postrend".

To observe when lazy loading occurs, enable debugging by setting: this.$debugOpts.ddLazyjs = true;. This will provide detailed logs about the lazy loading process.


NOTICE:
will work → <script src="/assets/js/jq-test.js" dd-lazyjs="--after__rend"></script>
will also work → <script src="/assets/js/jq-test-module.js" type="module" dd-lazyjs="--after__rend"></script>
