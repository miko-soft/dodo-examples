dd-lazyjs

The attribute dd-lazyjs will load JS lazily i.e. after whole DOM is rendered, i.e. when all dd- elements are rendered.
To make this work just place dd-lazyjs attribute in the SCRIPT tag.

NOTIFY:
will not work → <script src="/assets/js/jq-test-module.js" type="module" dd-lazyjs></script>
will work → <script src="/assets/js/jq-test.js" dd-lazyjs></script>
