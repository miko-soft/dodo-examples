unloadAllCSS()


unloadAllCSS() removes all <link rel="stylesheet"> tags from <head> at once.
This example loads W3.CSS and a Google Font on page load.
Click the button to unload all stylesheets and observe the style changes. Open DevTools → Elements to watch the <head>.
Navigate away to restore — the __destroy() hook unloads only the CSS loaded by this example.







This text uses the Roboto Bold font loaded via Google Fonts.
