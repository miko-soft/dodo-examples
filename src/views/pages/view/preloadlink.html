preloadLink()


preloadLink() adds a <link rel="preload"> resource hint to <head>, telling the browser to fetch a resource early before it is needed.
The type argument maps to the as attribute: 'font', 'image', 'style', 'script', 'fetch', etc.
For type='font' the method automatically adds crossorigin="anonymous".
Duplicate URLs are silently ignored — calling preloadLink() twice with the same URL adds only one tag.


The following preloads are added automatically on page load:

  • preloadLink('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap', 'style')
  • preloadLink('/views/pages/home/main.html', 'fetch')

Open DevTools → Elements → <head> to see the injected <link rel="preload"> tags.
