loadCSS(), unloadCSS()


loadCSS() creates a <link rel="stylesheet"> tag and loads the CSS file.
unloadCSS() removes that tag and unloads the CSS. It is typically called in the __destroy() hook.
This example loads W3.CSS, which styles the buttons below.
The CSS is unloaded when the route changes, i.e. when the controller is destroyed. Open DevTools to observe.
