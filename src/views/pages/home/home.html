General Directives

  • dd-setinitial - sets the initial controller value from an element's value
  • dd-elem - captures a DOM element and stores it in the controller

    • WRITERS
  • dd-text - renders text from a controller property
  • dd-text method - renders text returned by a controller method
  • dd-text math - evaluates a math expression and renders the result
  • dd-text ternary - evaluates a ternary expression
  • dd-text concat - evaluates a string concatenation expression
  • dd-text-each - dd-text used inside a dd-each loop
  • dd-html - embeds raw HTML into the DOM

    • ATTRIBUTERS
  • dd-if , dd-elseif , dd-else - renders an element when the condition is truthy
  • dd-if expression - conditional rendering using an expression, dd-if="(this.a > 6)"
  • dd-if expression array - conditional rendering using an expression with an array of objects, dd-if="($model.companie[0].opened)"
  • dd-if groups - two if-groups sharing the same parent element
  • dd-if (nested) - an if-group nested inside another if element

  • dd-visible - shows or hides elements using the CSS visibility property

  • dd-value - sets an element's value from a controller property
  • dd-disabled - disables an element
  • dd-checked (checkboxes) - checks checkboxes using a controller property
  • dd-checked (radios) - checks radio buttons using a controller property
  • dd-selected - sets the selected attribute on single or multiple select elements
  • dd-class - modifies the class attribute
  • dd-style - modifies the style attribute
  • dd-src - sets the src attribute
  • dd-attr - sets any element attribute
  • dd-readonly - makes an element read-only (elem.readOnly)
  • dd-required - marks a form field as required (elem.required)
  • dd-placeholder - sets the placeholder attribute
  • dd-title - sets the title attribute (tooltip)
  • dd-data - sets a data-* attribute using the --suffix option
  • dd-min - sets the min attribute on input elements
  • dd-max - sets the max attribute on input elements

Listener Directives

  • dd-href - navigates to another page
  • dd-click - click event listener
  • dd-keyup - keyup event listener
  • dd-change - change event listener
  • dd-evt - chains multiple event listeners on one element
  • dd-set - sets a controller property from an input element (one-way binding)
  • dd-model - two-way binding with various form elements: input, checkbox, radio, select, textarea
  • dd-model & dd-set - using dd-model as a replacement for the dd-set + dd-value combination
  • dd-outclick - fires when the user clicks outside the element; listener attached to document, cleaned up automatically
  • dd-intersect - fires when the element enters the viewport via IntersectionObserver; supports --once and --threshold:N
  • dd-swipe - detects touch swipe gestures; filter by --left/--right/--up/--down or omit for any direction
