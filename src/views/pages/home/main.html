DoDo Framework Examples

Learn DoDo Framework through examples

  • Controller Lifecycle Hooks - shows in which order will be executed constructor, loader, init, postrender, destroy
  • Model - how to use model, a reactive variable designed with the Proxy Object
  • View::ddInc() - test the ddInc() method i.e. data-dd-inc attribute
  • View::loadViews() - test the loadViews() method with isAsync argument, i.e. data-dd-view attribute
  • View::lazyJS() - load/unload JS file lazily
  • DataDd - test elements with data-dd- attribute related to DataDd methods
  • DataDdListeners - test the data-dd- elements related to DataDdListeners methods
  • Cookie - test all Cookie methods: put, putObject, getAll, ...
  • BrowserStorage - test localStorage & sessionStorage
  • Form - test all Form methods: setControl, getControl, delControl ...
  • Auth [Login Form] - test login, logout, ... i.e. authentication methods
  • navig - test Navig (navigation) methods
  • Paginator - test pagination methods
  • antiflick - hide elements before the page is displayed, prevent flickering
  • i18n - test internationalization i.e. automatic translation to different languages
