Login (corelib/Auth)

The login test i.e. test the Auth core library.

1) Click /lib/auth/developer/dashboard to test Auth::isLogged() and Auth::hasRole() route guards.
If the user is already logged and has 'customer' role then the dashboard page will open. If not the route should redirect to the afterBadlogin URL.

2) To test the method Auth::autoLogin() just reload this URL. If the user is logged redirect to afterGoodLogin URL.
