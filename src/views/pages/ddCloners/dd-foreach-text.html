dd-foreach with dd-text

This is example where dd-text is used within or in same element where is dd-foreach directive. NOTICE: The option --forceRender is used. It tells framework to render the dd- directive every time render() method is trigered when the $model is changed. Explanation: When $model.ids is changed it will render directives with $model.ids and directives with some JS expression (...). Besides that the framework will reder direcitves with --forceRender directive.
