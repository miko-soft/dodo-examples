dd-foreach & dd-mustache


RECOMMENDATION: Use string interpolator ${...} instead of mustache {{...}} within the dd-foreach element.

A) dd-mustache as child of the dd-foreach element

{{${key} + 1}}. ${val.name}



B) dd-mustache and dd-foreach in the same element

{{${key} + 1}}. ${val.name}



NOTICE: Both examples give same result.
