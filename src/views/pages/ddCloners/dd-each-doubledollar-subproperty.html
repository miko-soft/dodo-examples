dd-each Doubledollar with subproperty

This example will show how to assign subproperty of double dollar value by using dd-set.
dd-each="companies --company,key" dd-set="$$company.note"

name size note
{{company.name}} {{company.size}}




Also dd-model can be used.
dd-each="$model.companies --company,key" dd-model="$$company.note"

name size note
{{company.name}} {{company.size}}
