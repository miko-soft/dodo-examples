dd-each2 complex

Demonstrates nested iteration using dd-each2 to loop over a sub-array within each outer dd-each row.
Use this pattern when each list item owns a collection of related records — such as a user with multiple companies. 

this.users = [
  {
    first_name: 'Peter',
    companies: [
      { name: 'Microsoft', address: '1 Microsoft Way, Redmond, WA' },
      { name: 'IBM', address: '1 New Orchard Rd, Armonk, NY' }
    ]
  },
  {
    first_name: 'John',
    companies: [
      { name: 'Google', address: '123 Main St, Anytown, USA' },
      { name: 'Yahoo', address: '123 Main St, Anytown, USA' }
    ]
  }
];

Usage:
dd-each="users --user,ui" — outer loop over users
dd-each2="user.companies --company,ci" — inner loop over each user's companies

# Name Address Companies
{{ui}} {{user.name}} {{ci}}: {{company.name}}  {{ci}}: {{company.address}}
