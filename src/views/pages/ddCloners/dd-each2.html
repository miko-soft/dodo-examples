dd-each2

Iterate a nested sub-array that belongs to each outer dd-each item. 

this.users = [
  { name: 'Peter', companies: ['Microsoft', 'IBM'] },
  { name: 'John',  companies: ['Google', 'Yahoo']  }
]

Usage:
dd-each="users --user,ui" — outer loop over users
dd-each2="user.companies --company,ci" — inner loop over each user's companies

# Name Companies
{{ui}} {{user.name}} {{ci}}:{{company}}
