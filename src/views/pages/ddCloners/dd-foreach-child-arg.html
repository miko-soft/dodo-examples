dd-foreach-child-arg

Let the array have objects of companies. Each object has a property with an array called 'workers': [{name, workers: [...]}, ...]
The dd-foreach should first iterate through the companies, and after that dd-foreach-child through workers.

The equivalent code will be: 
this.companies.forEach(company => {

  company.workers.forEach(worker => {

  });

});
