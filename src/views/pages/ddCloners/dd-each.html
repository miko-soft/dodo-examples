dd-each

Generate multiple HTML elements based on an array stored in a controller property.

Controller property

Example with pure controller property in the attribute value:
dd-each="names --val,key" AND dd-text="printKey({{key}})"

{{val}}


Model

Example with pure model in the attribute value.
Initial model value is undefined so cloned elements are not displayed.
The this. will not cause error.
dd-each="$model.companies --company,ind"

{{company.name}} - {{company.size}}
