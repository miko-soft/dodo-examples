dd-each

Generates multiple HTML elements from an array stored in a controller property.

Controller property

Example using a controller property directly as the attribute value:
dd-each="names --val,key" AND dd-text="printKey({{key}})"

{{val}}


Model

Example using a model property as the attribute value.
The initial model value is undefined, so no elements are rendered.
Using the this. prefix will not cause an error.
dd-each="$model.companies --company,ind"

{{company.name}} - {{company.size}}




Controller method
