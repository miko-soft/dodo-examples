dd-foreach-child

The dd-foreach-child element is nested within dd-foreach.
Used: ${company[field]} where "company" is array of objects and "field" is array of strings

Code

<table>
    <tr>
      <th dd-foreach="fields --field">${field}</th>
    </tr>
    <tr dd-foreach="$model.companies --company">
      <td dd-foreach-child="this.fields --field">${company[field]}</td>
    </tr>
  </table>
NOTICE: Use this.fields instead of fields because it is controller property.
In dd-foreach it s not required to use this, but in dd-foreach-child it must be used to distinguish controller property from the forEach() argument.

Example with $model

${field}
${company[field]}
