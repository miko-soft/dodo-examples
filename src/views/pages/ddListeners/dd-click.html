dd-click

Click event listener.

DD-CLICK — executes a controller method

Click the button to fetch JSON from the server.
DATA:

- runs a function defined as a controller property, e.g. print.inConsole.makeRed($element)

- tests a controller property as a function argument, e.g. dd-click="runCLICK2(this.products)"
- tests a controller property as a function argument, e.g. dd-click="runCLICK2(this.products[1].name)"

- tests various argument types: string, number, boolean, and regex

- click on the input field

- runs two controller methods
