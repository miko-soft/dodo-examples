dd-keyup

The keyup event listener.

DD-KEYUP (controller method)

Focus the INPUT field, type some letters and press enter. (Open console to watch the result.)
- bind to all keyup event codes
- bind to "enter" keyup event.code

DD-KEYUP (expression)

Type number and press enter. The SPAN will be filled with text from $element.value.
dd-elem="myElement" dd-keyup="(this.$model.ass_num1 = this.$elem.myElement.value) --enter" → $model.ass_num1 =
