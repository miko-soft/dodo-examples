dd-evt

Chains multiple event listeners on a single element.

Example 1

Multiple event listeners chained on one element.

dd-evt=" runEVT($element, $event, 'red') --mouseenter && runEVT($element, $event, 'green') --mouseleave && runEVT($element, $event, 'blue') --click " →
