dd-intersect

Execute a controller method when an element enters the browser viewport, powered by the native IntersectionObserver API — no scroll-event polling needed. Observers are stored in $dd.observers and disconnected automatically by ddUNLISTEN().

Options:
  • --once — disconnect the observer after the first intersection (fire-once)
  • --threshold:N — visibility ratio 0–1 required to trigger; default is 0 (any pixel)
