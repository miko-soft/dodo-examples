dd-href

Href listeners and changing URLs (browser history states).
NOTICE: The click on dd-href element will destroy current controller i.e. ddUNLISTEN() will be invoked.

LINK1 -- href="/" - This will reload the page because no dd-href attribute.
LINK2 -- href="/" dd-href - This will not reload the page and will act as single page app link.
LINK3 -- href="" dd-href="homePageURL" - This will not reload the page and will act as single page app link. The attribute value is controller property.
LINK4 -- href="" dd-href="(this.ddElemURL)" - This will not reload the page and will act as single page app link. The attribute value is JS expression.
LINK5 -- href="" dd-href="(`/dd/${this.partOfURL}`)" - This will not reload the page and will act as single page app link. The attribute value is JS expression.
