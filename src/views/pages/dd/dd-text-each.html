dd-text

Directive dd-text in combination with dd-each.

dd-text and dd-each in the same element



In most cases you should use doubledollar sign $$ to access the current value within dd-each.
dd-each="animals --val,key" dd-text="$$val"




But also you can use other ways to access the current value within dd-each:
dd-each="animals --val,key" dd-text="animals[{{key}}]"


dd-each="animals --val,key" dd-text="animals.{{key}}"


dd-each="animals --val,key" dd-text="printAnimal({{key}})"


dd-each="animals --val,key" dd-text="printAnimal2(animals[{{key}}], {{key}})"


dd-each="animals --val,key" dd-text="printAnimal2(animals.{{key}}, {{key}})"




dd-text element inside dd-each element

dd-each="animals --val,key" > dd-text="printAnimal($$val, {{key}})"


dd-each with template placeholders {{ ... }}

In this example the mustaches are used instead of dd-text: <p dd-each="animals --val,key">{{key}}. {{val}}</p>
