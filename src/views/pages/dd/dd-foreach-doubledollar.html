dd-foreach (doubledollar)

This is dd-foreach example where a double dollar is used.

While string interpolation ${val} will be replaced with iteration val value, the double dollar $$val will be replaced with val name (for example: $$company will be replaced with $model.companies[0]).

Code


  <table>
    <tr dd-foreach="$model.companies --company,key">
      <td>${company.name}</td>
      <td>${company.size}</td>
      <td><button type="button" dd-click="selectCompany()">select</button></td>
    </tr>
  </table>
