dd-selected

Sets the selected attribute using a controller property value.
SELECT-MULTIPLE: the controller value must be an array of strings.
SELECT-ONE: the controller value must be a string.

Controller property (select-multiple)


dd-selected="$model.user_names --multiple" →



Controller property (select-one)


dd-selected="$model.positions" →



Controller method (select-one)


dd-selected="setAnimal('dog')" →
