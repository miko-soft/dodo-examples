dd-selected

Set the "selected" attribute with the controller property value.
SELECT-MULTIPLE → controller value must be array of strings
SELECT-ONE → controller value must be a string

Controller property (select-multiple)


dd-selected="$model.user_names --multiple" →



Controller property (select-one)


dd-selected="$model.positions" →



Controller method (select-one)


dd-selected="setAnimal('dog')" →
