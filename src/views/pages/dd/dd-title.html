dd-title

Sets the title attribute (tooltip) on any element from a controller property or method.
Hover over the elements to see the tooltip.

Controller property

Example using a controller property directly as the attribute value.
dd-title="myTitle" →

Model

Example using a model property as the attribute value.
dd-title="$model.tip" →
dd-title="this.$model.tip" → Hover this text (model)
 

Controller method

Tooltip text returned by a controller method.
dd-title="getTitle()" →

Works on any element

dd-title can be applied to any HTML element.
