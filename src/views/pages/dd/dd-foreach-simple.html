dd-foreach Simple Example

dd-foreach properties

  • define template variables with option --val,key
  • use string interpolations ${}
  • use JS functions inside string interpolation, for example: ${Math.random() + key} or ${JSON.stringify(val)}
  • use controller properties in the string interpolation, ${key + this.someNumber} or ${val.name + ' is the name'} → NOTICE: this is related to controller instance


Example: (open browser console)

${key + this.someNumber}. | ${customer.id} | ${customer['name']} | ${customer.age} | ${customer.active} |
${JSON.stringify(customer)}
