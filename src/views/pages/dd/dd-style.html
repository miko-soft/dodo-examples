dd-style

Modify the "style" attribute with the controller property value. Controller property must be object or null.

Controller property



dd-style="$model.myStajl" → "font-weight:bold" will remain
dd-style="$model.myStajl --replace" → "font-weight:bold" will be removed


Controller method

style="font-style: italic;" dd-style="setStyle() --replace" →

font: '13px Verdana', color: 'orangered' (font-style: italic is removed due to --replace)
