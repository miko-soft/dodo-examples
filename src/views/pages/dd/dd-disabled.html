dd-disabled

Disables elements by setting the disabled attribute.

Controller property

Example using a controller property directly as the attribute value.
dd-disabled="myBool" →

Model

Example using a model property as the attribute value.
The initial model value is undefined, so the element is disabled.
Using the this. prefix will not cause an error.
dd-disabled="$model.my.bool" →
dd-disabled="this.$model.my.bool" →
