dd-disabled

Disable elements by setting up disabled attribute.

Controller property

Example with pure controller property in the attribute value.
dd-disabled="myBool" →

Model

Example with pure model in the attribute value.
Initial model value is undefined so element is disabled.
The this. will not cause error.
dd-disabled="$model.my.bool" →
dd-disabled="this.$model.my.bool" →
