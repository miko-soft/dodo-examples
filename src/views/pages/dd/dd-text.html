dd-text

Print pure text.

Overwrite contentText

dd-text="product.name" → PLACEHOLDER (notice: style will stay on the element unchanged)


this.

If prefix this. is used it will not cause the error.
dd-text="this.product.name" →


HTML

The HTML tags will not be parsed.
dd-text="textWithHtml" →


Pipe option --pipe:

The pipe is any JS String function.
dd-text="longText"→
dd-text="longText --pipe:slice(0, 10).replace(/ipsu/i, 'IPSU').trim()" →
dd-text="longText --pipe:replace(/Lorem/ig, 'LOREM')" →


Automatic data type conversion:

The data type will be automatically converted to string: object → JSON string, boolean → string, number → string
dd-text="obj4json" convert object to JSON string →
dd-text="bool" convert boolean to string →


Initial value is undefined

If the initial controller's value is undefined or null the element will not be cloned.
