dd-text

Renders plain text content.

Overwrite text content

dd-text="product.name" → PLACEHOLDER (notice: style will stay on the element unchanged)


this.

Using the this. prefix will not cause an error.
dd-text="this.product.name" →


HTML

HTML tags will not be parsed — they are rendered as plain text.
dd-text="textWithHtml" →


Pipe option --pipe:

The pipe accepts any JavaScript String method.
dd-text="longText"→
dd-text="longText --pipe:slice(0, 10).replace(/ipsu/i, 'IPSU').trim()" →
dd-text="longText --pipe:replace(/Lorem/ig, 'LOREM')" →


Automatic data type conversion:

Data types are automatically converted to strings: objects become JSON strings, booleans and numbers become their string representations.
dd-text="obj4json" convert object to JSON string →
dd-text="bool" convert boolean to string →


Initial value is undefined

If the controller value is initially undefined or null, the element will not be rendered.
