dd-data

Sets a data-* attribute on an element. Use the --suffix option to specify the data attribute name.

Controller property

Sets data-id using a controller property.
dd-data="myId --id" → element
Inspect the element to see data-id="user-42"

Model

Sets data-category from a model property.
dd-data="$model.category --category" →
product card



Sets data-item-id from a model property.
dd-data="$model.itemId --item-id" →


Controller method

Sets data-role using a controller method.
dd-data="getRole() --role" → role element
Inspect the element to see data-role="admin"

Reading data attributes via JavaScript

Data attributes set by dd-data are accessible via dataset.
