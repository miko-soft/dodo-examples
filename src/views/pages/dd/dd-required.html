dd-required

Marks form fields as required by setting the required property (elem.required = true/false).

Controller property

Example using a controller property directly as the attribute value.
dd-required="myBool" →

Model

Example using a model property as the attribute value.
The initial model value is undefined, so the element is required.
Using the this. prefix will not cause an error.
dd-required="$model.req.bool" →
dd-required="this.$model.req.bool" →
