dd-if, dd-elseif, dd-else

Create an element when the if statement is truthy.

Controller property

Example with pure controller property in the attribute value.
dd-if="myBool" ; dd-else → OK NOT OK

Model

Example with pure model in the attribute value. Initial model value is undefined so dd-else is shown. The this. will not cause error.
NOTICE: if group is separated with SPAN tag from other if groups.
dd-if="$model.my.bool" ; else → OK NOT OK


Controller method

The controller method should return boolean or thruthy/falsy value.
dd-if="x_equal(2)" ; dd-elseif="x_equal(5)" ; dd-else → 2 5 not 2, not 5

Expression with model

Some complex examples with model in the expression.
dd-if="check_numbers('A')" | dd-elseif="check_numbers('B')" | dd-elseif="check_numbers('C')" | else → n1 > 6 and n2 > 10 n1 < 6 n2 < 10 ELSE
