dd-if, dd-elseif, dd-else

Create an element when the if statement is truthy.

Controller property

Example with pure controller property in the attribute value.
dd-if="myBool" ; dd-else → OK NOT OK

Model

Example with pure model in the attribute value. Initial model value is undefined so dd-else is cloned. The this. will not cause error.
NOTICE: if group is separated with SPAN tag from other if groups.
dd-if="$model.my.bool" ; else → OK NOT OK

Expression

Expression must be enclosed in brackets. This example shows expression with static controller property.
dd-if="(this.x === 2)" ; dd-elseif="(this.x === 5)" ; dd-else → 2 5 not 2, not 5

Expression with model

Some complex examples with model in the expression.
if(this.$model.n1 > 6 && this.$model.n2 > 10) | elseif(this.$model.n1 < 6) | elseif(this.$model.n2 < 10) → n1 > 6 and n2 > 10 n1 < 6 n2 < 10 ELSE
