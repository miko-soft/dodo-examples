dd-max

Sets the max attribute on <input> elements (number, range, date, time, etc.).

Controller property

Sets the maximum value of a number input using a controller property.
dd-max="myMax" (max=100) →

Model — number input

The maximum value is driven by the model.
dd-max="$model.maxVal" →
dd-max="this.$model.maxVal" →
 

Model — range input

Range slider with a dynamic maximum.
dd-max="$model.maxVal" →

Model — date input

Date picker with a maximum date from the model.
dd-max="$model.maxDate" (max=2025-12-31) →

Controller method

Maximum date returned by a method.
dd-max="getMaxDate()" (max=2026-12-31) →

Combined min + max

Use dd-min and dd-max together on the same element.

dd-min="myMin" (5) & dd-max="myMax" (100)
