Negation - !

!$model.companies[0].opened


Double Negation - !!

!!$model.companies[1].opened


Variable value

$model.companies[1].opened


Strict equal - ===

$model.companies[1]._id === 2


Equal - ==

$model.companies[1]._id == '2'


Greater or equal than - >=

$model.companies[1].rate >= $model.companies[0].rate


Greater than - >

$model.companies[1].rate > 50


Less or equal than - <=

$model.companies[0].rate <= 0


Less than - <

$model.companies[0].rate < 0


Not equal value or not equal type - !==

$model.companies[1].rate !== $model.companies[0].rate

$model.companies[1].rate != $model.companies[0].rate
