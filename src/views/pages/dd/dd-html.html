If the initial controller's value is undefined or null the element will not be generated at all.

Model

Ordinary controller property should be initialised in the __init() hook. For example: this.myHTML = '<b>bold txt</b>'
Model property should be initialised in the __postrend() hook to prevent double rendering. For example: this.$model.myHTML = '<b>bold txt</b>'
dd-html="$model.myHTML" →
