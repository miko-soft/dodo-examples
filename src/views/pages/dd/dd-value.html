dd-value

This directive takes controller property value or controller method returned value and sets the element value.


INPUT, TEXTAREA


undefined in input (text type) -- dd-value="$model.input_text_undefined" →
string in input (text type) -- dd-value="$model.input_text01" →
string in input (number type) -- dd-value="$model.input_numberAsString" →
object in input -- dd-value="$model.input_text_obj" →
object in textarea -- dd-value="$model.input_text_obj" →
