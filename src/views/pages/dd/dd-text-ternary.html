dd-text ternary condition

Solve a ternary condition. For example dd-text="(this.isActive ? 'YES' : 'NO')".


Initial variables: 
  this.isActive = false;
  this.a = 10;
  this.$model.b = 20;
  this.product = { size: 25 };


Strings

Examples where second and third operators are pure strings.

isActive ? 'YES' : 'NO' →
!isActive ? 'YES' : 'NO' →

a > 3 ? {color: 'green'} : {color: 'red'} →

$model.b <= 3 ? 'YES' : 'NO' →
$model.b <= product.size ? 'YES' : 'NO' →

true <= product.size ? 'YES' : 'NO' →
false <= product.size ? 'YES' : 'NO' →

Controller properties

Examples where second and/or third operator is controller property.

isActive ? product.size : 'NO' →
isActive ? product.size : $model.b →
!isActive ? product.size : $model.b →
