dd-placeholder

Sets the placeholder attribute on input/textarea elements from a controller property or method.

Controller property

Example using a controller property directly as the attribute value.
dd-placeholder="myPlaceholder" →

Model

Example using a model property as the attribute value.
dd-placeholder="$model.ph" →
dd-placeholder="this.$model.ph" →
 

Controller method

Placeholder text returned by a controller method.
dd-placeholder="getPlaceholder()" →

Textarea

Works on textarea elements too.
dd-placeholder="$model.ph" →
