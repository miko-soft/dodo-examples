dd-readonly

Makes elements read-only by setting the readOnly property (elem.readOnly = true/false).

Controller property

Example using a controller property directly as the attribute value.
dd-readonly="myBool" →

Model

Example using a model property as the attribute value.
The initial model value is undefined, so the element is read-only.
Using the this. prefix will not cause an error.
dd-readonly="$model.ro.bool" →
dd-readonly="this.$model.ro.bool" →
