dd-setinitial

Get the element value and set the initial controller property value (when the controller is executed). The element is input, textarea or select tag.

-- open console to see the results.

INPUT

--- dd-setinitial="years_old" [ NOTICE:incremented in __init() ] →
--- dd-setinitial="$model.isValid --convertType" →

TEXTAREA

--- dd-setinitial="my_text" →
--- dd-setinitial="$model.my_num --convertType" →

SELECT

--- dd-setinitial="my_select" →
