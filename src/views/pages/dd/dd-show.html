dd-show

dd-show="ctrlProp" → Show/hide elements by setting up display:none inline CSS style.
dd-show="ctrlProp --visibility" → Show/hide elements by setting up visibility:visible inline CSS style.

Controller property

Example with pure controller property in the attribute value.
dd-show="myBool" → OK
dd-show="this.myBool" → OK

Model

Example with pure model in the attribute value.
Initial model value is undefined so element is not displayed.
The this. will not cause error.
dd-show="$model.my.bool" OK
dd-show="this.$model.my.bool" OK


Expression

Expression must be enclosed in brackets. This example shows expression with static controller property.
(this.x === 'This is X') OK

Expression with model

Some complex examples with model in the expression.
(this.$model.n1 > 6 && this.$model.n2 < 10) OK
(this.$model.n1 > 6 && this.$model.n1 < 10) OK
