dd-foreach & dd-mustache

If dd-foreach contains dd-mustache it will cause the error.
Use string interpolator ${...} instead of mustache {{...}} in dd-repeat element and its childrens.

A) dd-mustache inside dd-foreach element

This will cause the error! Open console to see the error

{{${key} + 1}}. ${val.name}



B) dd-mustache and dd-foreach in the same element

This will cause the error! Open console to see the error

{{${key} + 1}}. ${val.name}
