dd-foreach with dd-mustache



A) dd-mustache inside dd-foreach element

Use it most often.

{{${key} + 1}}. ${val.name}



B) dd-mustache and dd-foreach in the same element

{{${key} + 1}}. ${val.name}



C) dd-mustache is wrapping dd-foreach

NOTICE 1: The class can be written with mustache class="kolor_{{${key1} + 1}}" or simplier class="kolor_${key1 + 1}"
NOTICE 2: The mustache can contain string interpolation ${...} {{ ${...} }}, for example: {{${key1} + 1}} what is equivalent to ${key1 + 1}

{{${key1} + 1}}. ${val1.name} - ${val1.size} - {{this.$model.myText}}



D) dd-mustache in OPTION

- generate SELECT/OPTION by dd-foreach and dd-mustache and set the default value
