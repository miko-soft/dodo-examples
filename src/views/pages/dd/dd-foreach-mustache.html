dd-foreach and dd-mustache

The element dd-mustache shouldn't be in same dd-foreach element or as chilc of dd-foreach.
Use string interpolator ${...} instead of mustache {{...}}.

A) dd-mustache inside dd-foreach element

This will cause the error! Open console to see the error

{{${key} + 1}}. ${val.name}



B) dd-mustache and dd-foreach in the same element

This will cause the error! Open console to see the error

{{${key} + 1}}. ${val.name}
