dd-foreach with dd-mustache



A) dd-mustache inside dd-foreach element

Use it most often.

{{${key} + 1}}. ${val.name}



B) dd-mustache and dd-foreach in the same element

{{${key} + 1}}. ${val.name}



C) dd-mustache is wrapping dd-foreach

Use it rarely because it has double elements than in the case A).

{{${key1} + 1}} ${val1.name} - ${val1.size} - {{this.$model.myText}}
