dd-visible

dd-visible="ctrlProp" → Show/hide elements by setting up visibility:visible|hidden inline CSS style.

Controller property

Example with pure controller property in the attribute value.
dd-visible="myBool" → OK
dd-visible="this.myBool" → OK

Model

Example with pure model in the attribute value. Initial model value is undefined so element is not displayed. The this. will not cause error.
dd-visible="$model.my.bool" OK
dd-visible="this.$model.my.bool" OK


Expression

Expression must be enclosed in brackets. This example shows expression with static controller property.
(this.x === 'This is X') OK

Expression with model

Some complex examples with model in the expression.
(this.$model.n1 > 6 && this.$model.n2 < 10) OK


Toggle

Toggle visibility with the button.
dd-visible="$model.visX"
