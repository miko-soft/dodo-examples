dd-visible

dd-visible="ctrlProp | ctrlMeth()" — Shows or hides elements by setting the CSS visibility property to visible or hidden.

Controller property

Example using a controller property directly as the attribute value.
dd-visible="myBool" → OK
dd-visible="this.myBool" → OK

Model

Example using a model property as the attribute value. The initial model value is undefined, so the element is not visible. Using the this. prefix will not cause an error.
dd-visible="$model.my.bool" OK
dd-visible="this.$model.my.bool" OK


Controller method

dd-visible="isVisible()" OK

Expression/condition

  $model.a <= 10 $model.a > 10

Toggle

Toggles visibility with the button.
dd-visible="$model.visX"
