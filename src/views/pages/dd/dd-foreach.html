dd-foreach

The dd-foreach will multiply element with the controller property value which should be array.

Controller property

Example with pure controller property in the attribute value.
dd-foreach="names --val,key" →

{{$$key + 1}} $$val



Model

Example with pure model in the attribute value.
Initial model value is undefined so cloned element is not displayed. (the undefined value is automatically convertd to empty array [] by the framework.)
The this. will not cause error.


dd-foreach="$model.companies --val1,key1" →

{{$$key1 + 1}} $$val1.name - $$val1.size
