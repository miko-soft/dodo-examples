dd-min

Sets the min attribute on <input> elements (number, range, date, time, etc.).

Controller property

Sets the minimum value of a number input using a controller property.
dd-min="myMin" (min=5) →

Model — number input

The minimum value is driven by the model.
dd-min="$model.minVal" →
dd-min="this.$model.minVal" →
 

Model — range input

Range slider with a dynamic minimum.
dd-min="$model.minVal" →

Model — date input

Date picker with a minimum date from the model.
dd-min="$model.minDate" (min=2024-01-01) →

Controller method

Minimum date returned by a method.
dd-min="getMinDate()" (min=2025-01-01) →
